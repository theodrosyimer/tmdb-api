# TMDB API

## Context

Formation Assofac DWWM 2022-2023
> Projet de site web pour le restaurant Chez Bella.

## Usage

**Install**

```sh
git clone https://github.com/theodrosyimer/tmdb-api
```

**Dev**

```sh
npm run dev
```

**Build**

```sh
npm run build
```

**Preview**

if the project is already built:

  ```sh
  npm run preview
```

otherwise do both:

```sh
  npm run build && npm run preview
```
